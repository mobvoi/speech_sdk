## 错误码说明

| 错误码 | 备注 |
| :---- | :---- |
| 10002 | 上传方式错误，需使用form表单方式 |
| 10003 | 文件超出512K限制 |
| 10004 | appkey、signature、timestamp参数不全 |
| 10005 | 没找到上传的音频文件 |
| 10006 | 鉴权失败（原因可能是appkey非法或者signature计算错误） |
| 31000 | 参数错误，缺少必填参数或者参数取值范围错误，具体可参考errorMessage中描述|
| 31002 | 服务器内部错误 |
| 31006 | 服务器并发量超出限制 |
